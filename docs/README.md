# Desert Survivors - Landing Page

A showcase landing page for **Desert Survivors**, a Vampire Survivors-style roguelike survival game set in the mythical Arabian desert.

## View the Page

**[Visit Landing Page](https://desert-survivors.vercel.app/)** | [GitHub Pages Mirror](https://hamz1188.github.io/DesertSurvivors/)

## About This Page

This landing page showcases the iOS game with:
- App icon and branding
- Feature highlights
- Character roster (8 playable characters)
- Weapon gallery (12 weapons + evolutions)
- Game statistics
- Coming Soon to App Store call-to-action

## The Full Game

The complete **Desert Survivors** iOS game features:

- **8 unique playable characters** with special abilities
- **12 weapons** with awakened evolution forms
- **8 enemy types** across 2 tiers with animated sprites
- **Procedurally generated** 4000x4000 desert world
- **Meta-progression** - permanent upgrades, character unlocks
- **Pixel art visuals** with PixelLab AI-generated assets

### Coming Soon to the App Store

The full iOS version is currently in development and will be available on the App Store soon.

## Technical Details

- **Engine**: Kaboom.js v3000
- **Single file**: All HTML, CSS, and JavaScript in one `index.html`
- **Mobile-friendly**: Responsive design with touch controls
- **iPhone mockup**: Game displayed in a realistic device frame

## Development

This demo was created as a portfolio piece to showcase the Desert Survivors concept. The full game is built natively in Swift using SpriteKit.
